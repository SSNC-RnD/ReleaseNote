# ReleaseNote

오아시스 릴리즈 노트 입니다.

※ 해당 릴리즈 노트는 기능 갱신에 따라 수시로 변경될 수 있습니다.


#작성 가이드 
각 파일에 기존 작성된 내역 상위에 다음과 같은 형태로 추가 합니다.

[날짜] [{add/mod/del}] [설명.....]

add - 기능 추가
       : 신규 기능이 추가되는 경우입니다.
mod - 기능 수정
       : 기존 작성되어 있는 기능이 변경되는 경우입니다.
del - 기능 삭제
       : 기존 작성되어 있는 기능이 삭제되는 경우 추가됩니다.

설명 : 메뉴트리를 입력후 추가된 기능을 기입 합니다. 
       ex)
       :  관리자 - 기타설정 - 로그인횟수 제한 
          로그인 횟수동안 실패 시 계정이 잠깁니다.
